# Examples

One file per language showing the Expected Load grammar in its native comment
syntax, plus a `frameworks/` folder with realistic scenarios.

These files are reference material, dedicated to the public domain (CC0) — copy
them freely.

| File | Language | Form shown |
| --- | --- | --- |
| [`terraform.tf`](./terraform.tf) | HCL / Terraform | block + inline |
| [`typescript.ts`](./typescript.ts) | TypeScript / JavaScript | JSDoc block + `//` inline |
| [`python.py`](./python.py) | Python | `#` block + docstring block |
| [`service.go`](./service.go) | Go | `//` block + `//directive` inline |
| [`Example.java`](./Example.java) | Java / Kotlin | Javadoc block |
| [`lib.rs`](./lib.rs) | Rust | `///` block |
| [`workload.yaml`](./workload.yaml) | YAML and other `#` formats | `#` block |
| [`frameworks/`](./frameworks) | — | end-to-end scenarios |
